Init with:
```
git clone https://github.com/new-programmist/Parses
cd Parses/
echo "alias parses='$PWD/parses.sh'" >> ~/.bashrc
source ~/.bashrc
chmod +x parses.sh
```

to test/run:

```
cd example
parses
```
